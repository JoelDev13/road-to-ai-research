#### Variables

Students learn how to:

- create variables
- update variables
- reuse variables

Example:

```python
age = 32
age += 1
```

#### Data Types

Main Python data types:

- int
- float
- string
- bool

#### Type Conversion

Implicit conversion.

Explicit conversion using:

```python
int()
float()
```

#### Constants

Constants should be written in uppercase.

Example:

```python
INCHES_IN_FOOT = 12
```

#### math Library

Introduces:

- math.sqrt()
- math.pi
- math.e
- math.log()

#### random Library

Random number generation:

```python
random.randint()
random.random()
random.uniform()
```

#### Functions

Students learn to split programs into reusable functions

Benefits:

- cleaner code
- easier maintenance
- code reuse

#### Variable Scope

Variables created inside a function are local to that function

#### While Loops

Syntax:

```python
while condition:
```

Repeat code while the condition is true

#### Boolean Values

Two values:

```python
True
False
```
#### If Statements

Decision making using:

```python
if
elif
else
```
